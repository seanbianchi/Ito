This repo is based off the party game "Ito" (if you've never heard of it and like board games I would highly recommend you play, not affilitated)

The categories can sometimes get stale or some are just boring

Add words one at a time or bulk paste
Tag words as NSFW and toggle adult content on/off
No-repeat mode so cards don't get drawn twice
Save and load multiple word lists
Export/import lists as .json to share with friends
Share lists via URL (doesn't work on iOS)
Scale labels — define what 1 and 100 mean for each category

Visit: https://seanbianchi.github.io/ito/
